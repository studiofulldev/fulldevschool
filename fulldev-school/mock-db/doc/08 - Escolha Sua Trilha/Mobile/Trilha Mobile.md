## Conteúdo

## Objetivo da trilha

Te levar até um app simples funcionando em ambiente mobile.

## Pré-requisitos

- base de lógica
- noção de interface

## Ordem sugerida

1. escolher nativo ou multiplataforma
2. aprender interface básica
3. navegação
4. estado simples
5. consumo de API
6. projeto inicial

## Ferramentas

- Android Studio ou stack equivalente
- Git

## Exercícios e projetos

- app de hábitos
- lista de compras

## Critério para avançar

Avance quando você conseguir montar tela, navegação e persistência simples.

## Erros comuns

- complicar ambiente cedo
- escolher stack só por hype
