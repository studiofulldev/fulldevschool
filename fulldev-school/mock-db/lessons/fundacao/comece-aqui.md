# 3 - Comece Aqui

## intro

Se você está chegando agora, a primeira coisa importante é esta:

entrar na área de tecnologia não começa escolhendo linguagem, framework ou salário.

## mapa-mental

Você precisa primeiro entender:

- o que é a área
- como ela funciona
- quais caminhos existem
- qual é o seu próximo passo
- como usar IA como ferramenta sem terceirizar completamente o seu crescimento

## mercado

O mercado existe, mas não é mágica.

Você vai precisar de base, prática, consistência e capacidade de usar ferramentas novas com critério.
