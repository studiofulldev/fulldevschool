# 4 - Fundamentos de Tecnologia

## mapa-basico

Nesta etapa, você monta um mapa mental do ecossistema tech.

Sem isso, a pessoa até segue tutorial, mas continua perdida quando escuta termos como API, frontend, backend, banco de dados e cloud.

## conceitos-centrais

Conceitos que você precisa reconhecer:

- hardware e software
- internet e web
- cliente e servidor
- API
- banco de dados
- versionamento

## por-que-isso-importa

Esses conceitos reduzem atrito.

Quando você entende o vocabulário, para de estudar no escuro e passa a conectar melhor o que cada ferramenta realmente faz.
