## Como usar esta etapa

Você não precisa fazer várias trilhas ao mesmo tempo.

O melhor é:

1. escolher uma trilha principal
2. seguir por algumas semanas
3. produzir algo pequeno
4. reavaliar com base em prática real

## Regra prática

Se você ainda está muito dividido, escolha a trilha que:

- parece mais estudável para você agora
- tem um primeiro projeto mais viável
- gera mais curiosidade do que resistência

## O que você deve sair sabendo ao terminar esta etapa

Ao concluir esta página, o ideal é que você tenha:

1. uma trilha principal escolhida
2. uma ordem inicial de estudo
3. uma noção clara do primeiro projeto possível
