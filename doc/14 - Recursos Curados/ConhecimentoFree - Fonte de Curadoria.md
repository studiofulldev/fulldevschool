# ConhecimentoFree - Fonte de Curadoria

O repositório [MrClaro/ConhecimentoFree](https://github.com/MrClaro/ConhecimentoFree) deve ser usado como **fonte de abastecimento** para descoberta de recursos, links e materiais úteis para o guia.

## Como usar esta fonte

- Procurar links oficiais, gratuitos e abertos
- Aproveitar listas de cursos, livros, documentações e desafios
- Validar manualmente cada link antes de promovê-lo ao guia principal

## Critério editorial

- Priorizar documentação oficial
- Priorizar plataformas conhecidas e legais
- Evitar replicar links sem contexto
- Evitar recursos de procedência duvidosa
- Transformar listas brutas em curadoria explicada

## O que vale aproveitar primeiro

- Repositórios de desafios técnicos
- Documentações oficiais
- Plataformas abertas de cursos
- Livros e materiais gratuitos reconhecidos
- Boas práticas de Git, README e colaboração

## Menção obrigatória

Ao usar essa base para alimentar o guia, mencionar que parte da descoberta de links foi inspirada no trabalho colaborativo do projeto `MrClaro/ConhecimentoFree`.
