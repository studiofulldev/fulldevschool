## Conteúdo

Use este painel para se localizar sem depender só de sensação.

## Checklist principal

- [ ] Entendi como a área funciona
- [ ] Organizei minha base digital
- [ ] Entendi os conceitos-base de tecnologia
- [ ] Montei uma rotina de estudo
- [ ] Escolhi uma área principal
- [ ] Escolhi uma trilha inicial
- [ ] Fiz meu primeiro projeto
- [ ] Criei ou organizei meu GitHub
- [ ] Montei currículo
- [ ] Configurei LinkedIn
- [ ] Fiz minhas primeiras candidaturas

## Como usar este painel

Não use este checklist para se cobrar de forma irracional.

Use para:

- saber onde você está
- ver o que já avançou
- identificar o próximo gargalo

## Próxima ação prática

Se alguma etapa estiver faltando, volte para ela e resolva uma coisa por vez.

### Como interpretar a taxa de assertividade

- `95% a 100%`: informação diretamente sustentada por documentação ou dado forte.
- `85% a 94%`: síntese confiável, mas com algum grau de generalização editorial.
- `70% a 84%`: orientação útil, porém mais interpretativa e dependente de contexto.
