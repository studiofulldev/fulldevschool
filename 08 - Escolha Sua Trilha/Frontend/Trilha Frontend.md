# 9.1 - Trilha Frontend

**Objetivo:** organizar uma sequência prática para quem decidiu começar por frontend.

**Passo anterior:** [[../Índice - Trilhas|9 - Escolha Sua Trilha]]

**Próximo passo:** [[../../09 - Projetos e Portfólio/Índice - Projetos e Portfólio|10 - Projetos e Portfólio]]

## Conteúdo

## Objetivo da trilha

Te levar da base web até um primeiro projeto visual publicável.

## Pré-requisitos

- computador organizado
- noção básica de internet e programação

## Ordem sugerida

1. HTML
2. CSS
3. JavaScript básico
4. DOM e interação
5. projeto simples
6. Git e GitHub
7. framework depois da base

## Ferramentas

- VS Code
- navegador
- Git
- GitHub

## Exercícios e projetos

- landing page
- lista de tarefas
- página com consumo simples de API

## Critério para avançar

Avance quando você conseguir montar algo simples sem copiar tudo passo a passo.

## Erros comuns

- querer React cedo demais
- decorar sem praticar
- ignorar CSS

## Recursos complementares

- MDN
- roadmap.sh/frontend
