## Conteúdo

## Objetivo da trilha

Te levar da base operacional até automação e entendimento inicial de cloud.

## Pré-requisitos

- fundamentos de tecnologia
- conforto com terminal ajuda

## Ordem sugerida

1. Linux básico
2. redes básicas
3. Git
4. Docker
5. CI/CD
6. cloud básica

## Ferramentas

- terminal
- Docker
- GitHub Actions

## Exercícios e projetos

- containerizar app
- automatizar pipeline simples

## Critério para avançar

Avance quando você entender o fluxo de build, deploy e ambiente.

## Erros comuns

- pular fundamentos
- acumular ferramenta sem contexto
