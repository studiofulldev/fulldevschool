## Conteúdo

## Objetivo da trilha

Te levar da base de lógica e web até uma API simples com persistência.

## Pré-requisitos

- fundamentos de tecnologia
- noção inicial de programação

## Ordem sugerida

1. escolher uma linguagem
2. lógica e sintaxe básica
3. HTTP e APIs
4. banco de dados
5. autenticação simples
6. projeto pequeno

## Ferramentas

- linguagem inicial
- Postman
- Git
- banco de dados

## Exercícios e projetos

- API de tarefas
- cadastro de usuários

## Critério para avançar

Avance quando você conseguir criar rota, validar dado e salvar informação sem depender de cópia total.

## Erros comuns

- estudar framework antes de entender HTTP
- ignorar banco de dados

## Recursos complementares

- roadmap.sh/backend
- MDN HTTP
